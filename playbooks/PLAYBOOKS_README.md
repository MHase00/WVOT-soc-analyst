# Playbooks & Documentation

This directory contains operational playbooks and procedures for WVOT SOC.

## Included Playbooks

1. **false-positive-tuning-5-step-process.md**
   - Methodology for reducing alert noise
   - Real-world case study (SQL injection example)
   - Step-by-step guide with expected outcomes

2. **incident-response-template.md**
   - Standardized template for incident response
   - Escalation procedures
   - Documentation requirements for audit readiness

## How to Use

1. Copy the template relevant to your use case
2. Fill in specific details for your incident/environment
3. Follow the procedures step-by-step
4. Document actions and outcomes
5. Archive for future reference and audit

## Contributing

To add new playbooks:
1. Create a new .md file
2. Follow the existing template structure
3. Include real examples where possible
4. Add to this README with a brief description
